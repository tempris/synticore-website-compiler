This is a minimal Synticore project used to build Browsersync-only development toolkit assets.

The toolkit contract and intended purpose are documented in source/project/browser/README.md and explicitly forbid layout-affecting debug behavior.

The watch overlay uses the built files at /asset/css/dev/dev-page.min.css, /asset/css/dev/dev-panel.min.css, and /asset/js/dev-overlay.min.js.

The fallback page for missing routes is built from in/404.html into out/404.html.
