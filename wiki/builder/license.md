## Practical Rule Of Thumb

When reusing files from the example project repository:

1. Read the current example-project license from the example repository first
2. Keep/reuse technical structure, templates, and code according to that license
3. Replace brand names/logos/icons with your own identity
4. Keep attribution optional unless your own project policy requires it

Optional attribution suggestions:

- "Built using Synticore Website Compiler by Cure Interactive."
- "Built using Synticore Website Compiler."
